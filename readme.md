# AI for Mental Health Prediction

This project uses Natural Language Processing and Machine Learning
to predict mental health indicators from text-based user input.

## Features
- Text preprocessing using NLP
- TF-IDF vectorization
- Logistic Regression classifier
- Streamlit web interface

## Technologies Used
- Python
- Scikit-learn
- NLTK
- Streamlit

## How to Run
1. Install dependencies
2. Train the model
3. Launch Streamlit app

## Disclaimer
This project is for academic purposes only and is not a medical tool.
